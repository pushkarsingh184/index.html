Overall architecture

SNPM consists of 4 modules: Fire Risk (regression), Poaching Risk (classification), Visitor Forecast (time-series regression), and Wildlife Detection (image classification via transfer learning).

Datasets

FileColumns / Description
synthetic_fire.csvtemperature, humidity, wind_speed, rainfall, veg_type, fire_risk (0-100)
synthetic_poaching.csvseason, zone_type, dist_to_village, past_incidents, patrol_freq, poaching_label (0/1)
synthetic_visitors.csvdate, day_of_week, holiday, bad_weather, visitors
wildlife_images/*folder per class: e.g. tiger/, deer/, elephant/ — used for transfer learning

Feature engineering & sample features

  • Fire: temperature, humidity, wind_speed, rainfall, veg_type (categorical encoded 0/1/2)
  • Poaching: season (cat), zone_type (cat), distance to village (numeric), past incidents (count), patrol_freq (cat)
  • Visitor: day_of_week, holiday flag, bad_weather flag, previous day visitors (lag feature)
  • Wildlife: images resized to 160×160, augmented (flip, rotate, shift) for transfer learning

Models & hyperparameters

ModuleModelKey hyperparams

PoachingRandomForestClassifiern_estimators=100, random_state=24
VisitorsRandomForestRegressorn_estimators=80, random_state=11, features include lag
WildlifeMobileNetV2 (transfer learning)base frozen, dense head 128 neurons, epochs=6 (default), batch=16

Training & evaluation

Typical pipeline:

  1. Split dataset train/test (80/20)
  2. Train RandomForest (or chosen model)
  3. Evaluate — Fire/Visitor: RMSE, Poaching: accuracy/precision/recall, Wildlife: accuracy / confusion matrix

Saved files

Models saved in /models (joblib for numeric models, .h5 for Keras wildlife model). App loads these files at runtime.

Sample training command & code (fire)

python src/generate_synthetic_data.py
python src/train_numeric_models.py
# results: models/fire_model.joblib

Notes for reproducibility

  • Set random_state for reproducible splits
  • Document dataset generation parameters (seeds, distributions)
  • For the wildlife model, ensure 50+ images per class for decent accuracy

How to run locally (quick)

1) Unzip the project folder and open snpm_project/src.
2) Install packages:
pip install streamlit pandas numpy scikit-learn matplotlib seaborn joblib tensorflow pillow
3) Run the dashboard:
streamlit run app.py
4) Open http://localhost:8501 in your browser.
